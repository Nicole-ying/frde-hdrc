# 交付包目录

## 交付定位

本交付包已按“无人工领域知识的 agent 自主奖励搜索”口径整理。主 FDRE-HRDC 路径不再把 LunarLander 的人工状态语义或手写奖励规则注入提示词，而是通过通用环境接口、原始奖励、训练日志、运行错误和评估分数进行迭代搜索。固定奖励程序仅保留为 legacy/canonical 对照和消融参考。

## 效果结论

当前效果建议按“两段式证据链”汇报：100k timesteps 用于说明快速奖励搜索、样本效率和消融差距；1M timesteps 用于说明最终收敛效果。1M 下 FDRE-HRDC 三个 seed 分别达到 223.40、219.98、234.74，全部超过 200 分 solved 标准，平均得分为 226.04±6.31，reward error 和奖励函数错误导致的训练中断均为 0。
