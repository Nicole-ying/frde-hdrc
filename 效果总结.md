# 当前实验效果总结

当前效果可以作为交付使用，核心结论应聚焦在 1M timesteps 的稳定收敛结果上。FDRE-HRDC 在 LunarLander-v3 上使用 PPO 进行策略训练，最终评估统一采用环境原始奖励，而不是 shaping 后的训练奖励；在 1M 设置下，三个随机种子分别达到 223.40、219.98 和 234.74，全部超过 LunarLander-v3 的 200 分 solved 标准，平均得分为 226.04，标准差为 6.31，最差 seed 也达到 219.98，说明方法不是依赖单次偶然高分，而是在多 seed 下稳定过线。同时，reward runtime error 为 0，由奖励函数错误导致的训练中断为 0，说明奖励函数生成、注入、运行检查和候选筛选流程是稳定可运行的。

100k timesteps 的结果不建议作为最终收敛分数来表述，而应作为快速搜索和消融验证来表述。在该设置下，FDRE-HRDC 达到 159.80±8.10，明显高于 PPO baseline 的 -4.57±41.13、单次 LLM 生成的 -58.01±51.44、去掉诊断反馈的 27.91±21.78 和去掉动态权重的 19.16±38.15。该组结果说明，在相同较短训练预算下，agent 自主迭代搜索能够更快找到有效奖励函数；消融结果也显示，诊断反馈和动态权重都会带来明显性能贡献。最终论文或汇报中建议采用“100k 展示搜索效率与模块贡献，1M 展示最终策略收敛和 solved 能力”的表述方式。
